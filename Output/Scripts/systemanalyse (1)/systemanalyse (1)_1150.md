Abgeleitetes Attribut (derived attribute ): Abgeleitete Attribute lassen sich aus anderen Attribute n berechnen. Sie dürfen
nicht direkt geändert werden. Abstrakte Klasse (abstract class ): Von einer abstrakten Klasse können keine Objekte erzeugt werden. Die abstrakte Klasse
spielt eine wichtige Rolle in Vererbungsstrukturen, wo sie die Gemeinsamkeiten einer Gruppe von →Unterklassen deﬁniert.
Damit eine abstrakte Klasse verwendet werden kann, muss von ihr zunächst eine Unterklasse abgeleitet werden.
