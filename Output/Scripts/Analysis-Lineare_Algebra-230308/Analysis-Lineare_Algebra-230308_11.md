Menge keine Rolle. Mit dieser Beschreibung kann nun auch feststellen, dass die Zahl 9ein Element der
Menge Aist, und die Zahl 7kein Element der Menge Aist. In mathematischer Notation beschreibt
man das so: 9∈A,7̸∈A.
